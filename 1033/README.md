# 1033. To Fill or Not to Fill

http://www.patest.cn/contests/pat-a-practise/1033

这题是 2012 年浙大计算机考研复试机试题之一。

从起点到终点一路上有 N 个加油站，计算最省钱的加油策略。

贪心模拟，用尽可能少的钱到尽可能远的地方。

对于贪心的题目，先从测试用例出发把流程过一遍，然后对各种情况进行归类、总结。
